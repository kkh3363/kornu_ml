- 명령어로 만들기
    - 가상환경을 추가하는 명령어는 아래와 같다
        
        > conda create -n 가상환경 이름 python=파이썬 버전
        
        conda create -n test_env python=3.9
        > 
    - 가상환경 목록
        
        > conda env list
        > 
    - 추가한 가상환경을 활성화
        
        > conda activate 가상환경 이름
        
        conda deactivate
        > 
    - 가상환경 패키지
        
        > pip install 패키지명
        
        pip list  
        conda list
        >
